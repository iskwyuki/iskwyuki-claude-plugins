## 採点運用

- **較正アンカー（易）**: CO-6（易・情報露出）を「これを見逃す構成は資産が壊れているとみなす」較正床としてきたが、会社ケースは凍結済みで再走不能。**公開ケース（PF-*）には現状『易』ケースが無く、較正床の補充が今後の課題**（Plans 4.2 追補）。
- **弁別ケース（難）**: CO-4 / CO-9 / CO-11（会社・凍結）、PF-2（公開・状態管理）がモデル・資産間の差を出す。
- 判定は 4 区分（検出 / 部分検出 / 見逃し / 追加指摘）。詳細は [`../PROTOCOL.md`](../PROTOCOL.md)。
